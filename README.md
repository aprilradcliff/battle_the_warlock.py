# Battle the Warlock – Python Text Adventure

A command-line adventure game written in Python.  
The player navigates a castle, collects magical items, and defeats the Warlock.

## Project Overview

This project demonstrates:

- Python fundamentals (loops, conditionals, functions)
- Dictionaries for structured data
- Lists for inventory management
- Input validation
- Game-state logic (win/lose conditions)
- Modular code organization

## How It Works

- The game world is defined using nested dictionaries.
- Players move between rooms using directional commands.
- Each room may contain a collectible item.
- The player must collect all six items before entering the Gatehouse.
- Entering without all items results in defeat.

## Technologies Used

- Python 3
- Command Line Interface (CLI)
